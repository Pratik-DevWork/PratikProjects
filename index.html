Introduction

Python is a widely used general-purpose, high level programming language. It was created by Guido van Rossum in 1991 and further developed by the Python Software Foundation. It was designed with an emphasis on code readability and its syntax allows programmers to express their concepts in fewer lines of code. Python is a programming language that lets you work quickly and integrate systems more efficiently. Python is a dynamic, interpreted (bytecode-compiled) language. There are no type declarations of the variables, parameters, functions or methods in source code.

It is used for:

  • Web Development (Server side)
  • Software Development
  • System Scripting
  • App Development

Identifiers

Identifier is a name used to identify a variable, function, class, module, etc. Identifiers should start with a character or underscore, then use a digit. The characters A-Z, a-z and underscore (_) and digits (0-9) can be used but no special character ( #, @, $, %, ! ) in Identifiers.

Example:

Valid Identifier

# Valid identifiers

my_variable = 10
_name = "Pratik"
value123 = 99

Invalid Identifier

# Invalid identifiers

1st_variable = 20  # Cannot start with a digit
my-variable = 30   # Hyphen is not allowed
my variable = 40   # Space is not allowed

Reserved Keywords

Reserved Keywords (also called "Keywords") have predefined meaning and syntax in the language. These keywords are used to develop programming instructions.

To check Keywords list:

# List of Python keywords

import keyword
print(keyword.kwlist)

Variable

Python variables are containers which store values. A variable is a name given to a memory location that stores data. It is used to hold values that can be changed during the execution of a program. In Python, variables are created by assigning a value to a name using the assignment operator (=). We don't need to declare variables before using them, as Python is dynamically typed. It is the basic unit of storage in a program.

Example:

# Variable declaration and assignment
# Integer variable
x = 10  
name = "Pratik"    # String variable
pi = 3.14          # Float

Comments

Comments are lines in the code that are ignored by the interpreter during the execution of the program. Comments enhance the readability of the code and help programmers to understand the code carefully.

There are three types of comments in Python.

  • Single line Comments (Using #)
  • Multiline Comments (Using multiple #)
  • Docstring Comments (Using Triple ''')

Example:

# Single line comment
x = 10  # This is a Variable
# Hi, I am                 
# Pratik Lokhande                 
# This is a multiline comment
"""This is a docstring comment"""

Data Types

Data types are used to define the type of data that can be stored in a variable. Python has several built-in data types, including:

Types of Data types

  • Numeric
  • String
  • List
  • Sets
  • Tuple
  • Dictionary
  • Boolean

I. Numeric

Numeric data types are used to represent numbers.

Python has three Numeric data types:

  • Int

    • hold signed integer value for non-limited length

  • Float

    • hold floating point value and up to 15 decimal places

  • Complex

    • hold complex numbers in the form of a + bj, where a is the real part and b is the imaginary part

II. String

String data type is used to represent a sequence of characters. Strings are enclosed in single quotes (' ') or double quotes (" "). Python supports Unicode, allowing for the representation of characters from various languages.

III. List

List data type is used to represent an ordered collection of items. Lists hold different data and lists are mutable, meaning their elements can be changed after creation. Lists are defined using square brackets ([ ]), and elements are separated by commas.

IV. Sets

Set data type is used to represent an unordered collection of unique items. Sets don't hold duplicate values and sets are mutable, meaning their elements can be changed after creation. Sets are defined using curly braces ({ }), and elements are separated by commas.

V. Tuple

Tuple data type is used to represent an ordered collection of items. Tuples are immutable, meaning their elements cannot be changed after creation. Data in Tuple is write-protected. Tuples are defined using parentheses ( ), and elements are separated by commas.

VI. Dictionary

Dictionary data type is used to represent a collection of key-value pairs. Dictionaries are unordered and mutable, meaning their elements can be changed after creation. Dictionaries are defined using curly braces ({ }), with each key-value pair separated by a colon (:).

VII. Boolean

Boolean data type is used to represent truth values. It can have two possible values: True or False. Booleans are often used in conditional statements and logical operations. It is denoted by class bool. Boolean values will be either True or False.

Note: True and False with capital 'T' and 'F' are valid Boolean and Both are Reserved Keywords

Example:

# Numeric data types
x = 10          # Integer
y = 3.14        # Float
z = 2 + 3j      # Complex number

# String data type
name = "Pratik"

# List data type
# Index exist in list
my_list = [1, 2, 3, "Python"]

# Tuple data type
my_tuple = (1, 2, 3)

# Set data type
my_set = {1, 2, 3}

# Dictionary data type
my_dict = {"name": "Pratik", "surname": "Lokhande", "class": 9, "age": 15}

# Boolean data type
is_active = True

Operators

Python operators in general are used to perform operations on the values and variables. These are standard symbols used for the purpose of logical and arithmetic operations.

OPERATORS: Are the special symbols. Eg: -, +, *, /, etc.

OPERAND: It is the value on which the operator is applied

I. Arithmetic Operators

Arithmetic operators are used to perform mathematical operations like addition, subtraction, multiplication and division.

  • +

    • Addition: Add two operands

  • -

    • Subtraction: Subtract two operands

  • *

    • Multiplication: Multiplies two operands

  • /

    • Division (Float): Divides first operand with second

  • //

    • Division (Floor): Divides first operand with second

  • %

    • Modulus: Return the remainder when the first operand is divided by the second

  • **

    • Exponentiation: Returns first raised to power second

Example:

# Arithmetic operators

a = 10
b = 5

# Addition
print(a + b)  # Output: 15

# Subtraction
print(a - b)  # Output: 5   

# Multiplication
print(a * b)  # Output: 50

# Division (Float)
print(a / b)  # Output: 2.0 

# Division (Floor)
print(a // b) # Output: 2   

# Modulus
print(a % b)  # Output: 0

# Exponentiation
print(a ** b) # Output: 100000

II. Precedence

Precedence of operators determines the order in which operations are performed in an expression. Operators with higher precedence are evaluated before those with lower precedence.

Precedence of Operators:

  • Parentheses ( ) - Highest Precedence
  • Exponentiation (**)
  • Bitwise NOT ( ~ )
  • Multiplication (*), Division (/), Floor Division (//), Modulus (%)
  • Addition (+), Subtraction (-)
  • Bitwise AND (&), Bitwise XOR (^)
  • Relational Operator: (>, <, ==, !=, >=, <= )
  • Assignment Operator: ( =, += , -=, *=, /=, %=, //=, **=, &=, /=, ^=, >>=, <<= )
  • Identity Operator: is, is not
  • Membership Operator: in, not in
  • Logical Operator: not
  • Logical Operator: or
  • Logical Operator: and

III. Relational Operator

Relational operators are used to compare two values and return a boolean result (True or False). These operators are commonly used in conditional statements and loops. These operators are also called Comparison Operators.

Types of Relational Operators:

  • ==

    • Equal to: Returns True if both operands are equal

  • !=

    • Not equal to: Returns True if both operands are not equal

  • >

    • Greater than: Returns True if the left operand is greater than the right operand

  • <

    • Less than: Returns True if the left operand is less than the right operand

  • >=

    • Greater than or equal to: Returns True if the left operand is greater than or equal to the right operand

  • <=

    • Less than or equal to: Returns True if the left operand is less than or equal to the right operand

Example:

# Relational operators

a = 10
b = 5

print(a == b)  # Output: False

# Not equal to
print(a != b)  # Output: True

# Greater than
print(a > b)  # Output: True

# Less than
print(a < b)  # Output: False

# Greater than or equal to
print(a >= b) # Output: True

# Less than or equal to
print(a <= b) # Output: False

IV. Assignment Operator

Assignment operators are used to assign values to variables. The basic assignment operator is '=', but there are several compound assignment operators that combine assignment with other operations.

Types of Assignment Operators:

  • =

    • Assigns the value of the right operand to the left operand

    syntax: x = y + z

  • +=

    • Adds the right operand to the left operand and assigns the result to the left operand

    syntax: a += b

    a = a + b

  • -=

    • Subtracts the right operand from the left operand and assigns the result to the left operand

    syntax: a -= b

    a = a - b

  • *=

    • Multiplies the left operand by the right operand and assigns the result to the left operand

    syntax: a *= b

    a = a * b

  • /=

    • Divides the left operand by the right operand and assigns the result to the left operand

    syntax: a /= b

    a = a / b

  • %=

    • Calculates the modulus of the left operand by the right operand and assigns the result to the left operand

    syntax: a %= b

    a = a % b

  • //=

    • Performs floor division on the left operand by the right operand and assigns the result to the left operand

    syntax: a //= b

    a = a // b

  • **=

    • Raises the left operand to the power of the right operand and assigns the result to the left operand

    syntax: a **= b

    a = a ** b

  • &=

    • Performs Bitwise AND on operand and assign value to left operand

    syntax: a &= b

    a = a & b

  • |=

    • Performs Bitwise OR on operand and assign value to left operand

    syntax: a |= b

    a = a | b

  • ^=

    • Performs Bitwise XOR on operand and assign value to left operand

    syntax: a ^= b

    a = a ^ b

  • <<=

    • Performs Bitwise Left Shift on operand and assign value to left operand

    syntax: a <<= b

    a = a << b

  • >>=

    • Performs Bitwise Right Shift on operand and assign value to left operand

    syntax: a >>= b

    a = a >> b

Example:

# Assignment operators

x = 5       # Assign 5 to x
y = 6       # Assign 6 to y

x += y      # x = x + y → 11
print(x)

x -= y      # x = 11 - 6 → 5
print(x)

x *= y      # x = 5 * 6 → 30
print(x)

x /= y      # x = 30 / 6 → 5.0
print(x)

x %= y      # x = 5.0 % 6 → 5.0 (remainder)
print(x)

x //= y     # x = 5.0 // 6 → 0.0 (floor division)
print(x)

x **= y     # x = 0.0 ** 6 → 0.0
print(x)

V. Membership Operators

in and not in are the membership operators; both are used to test whether a value or variable is in a sequence. It returns "True" and "False"

  • in: True if the variable is found in the sequence.
  • not in: True if the value is not found in the sequence.

Example:

# Example of membership operators

my_list = [1, 2, 3, 4, 5]

# Using 'in'
print(3 in my_list)          # Output: True

# Using 'not in'
print(10 not in my_list)     # Output: True

my_string = "Python"

print('P' in my_string)      # Output: True
print('z' not in my_string)  # Output: True

VI. Identity Operators

is and is not are the identity operators; both are used to check if two values are located on the same part of memory.

  • is: True if the operands are identical.
  • is not: True if the operands are not identical.

Example:

# Example of identity operators

a = [1, 2, 3]
b = a         # b references the same list as a
c = [1, 2, 3] # c is a separate list with the same content

# Using 'is'
print(a is b)   # True → a and b refer to the exact same object in memory
print(a is c)   # False → a and c have same values but are stored at different memory locations

# Using 'is not'
print(a is not c)  # True → a and c are not the same object in memory

Bitwise Operators

Bitwise operators in Python are special operators that work at the bit level — they perform operations directly on the binary (0s and 1s) representation of integers.

Types of Bitwise Operators

  • Bitwise AND (&)
  • Bitwise OR (|)
  • Bitwise XOR (^)
  • Bitwise NOT ( ~ )
  • Bitwise Left Shift (<<)
  • Bitwise Right Shift (>>)

Example:

a = 5   # Binary: 0101
b = 3   # Binary: 0011

# 1  → Binary: 0001 (AND)
print(a & b)

# 7  → Binary: 0111 (OR)
print(a | b)

# 6  → Binary: 0110 (XOR)
print(a ^ b)

# -6 → Inverts bits (two's complement form)         
print(~a)

# 10 → Binary: 1010 (Left shift by 1)
print(a << 1)

# 1 → Binary: 0001 (Right shift by 1)
print(b >> 1)

Expressions In Python

An expression is a combination of operands and operators that produces a value. It can be as simple as a single value or as complex as a combination of multiple values and operations.

Types of Expressions

  • Constant Expression

    • Contains only constant values

  • Variable Expression

    • Contains variables and their values

  • Assignment Expression

    • Assigns a value to a variable and returns the value

  • Conditional Expression

    • Evaluates a condition and returns one of two values based on the condition

  • Integral Expression

    • Contains integral values and operations

  • Floating Expression

    • Contains floating-point values and operations

  • Combination Expression

    • Combines multiple expressions using operators

  • Arithmetic Expression

    • Contains arithmetic operators and operands

  • Relational Expression

    • Contains relational operators and operands

  • Logical Expression

    • Contains logical operators and operands

  • Bitwise Expression

    • Contains bitwise operators and operands

Data Type Conversion

Data type conversion is the process of converting a value from one data type to another. In Python, this can be done using built-in functions such as int(), float(), str(), and others.

Types of Data Type Conversion

  • Implicit Conversion

    • Automatic conversion by Python without losing information

  • Explicit Conversion

    • Manual conversion using built-in functions

Example:

# Implicit Conversion
x = 10          # Integer
y = 3.14        # Float
z = x + y       # Implicitly converts x to float
print(z)        # Output: 13.14

# Explicit Conversion
a = "5"         # String
b = int(a)      # Converts string to integer
print(b)        # Output: 5
c = float(b)    # Converts integer to float
print(c)        # Output: 5.0

Input() Function

The input() function is used to take input from the user. It reads a line from input, converts it into a string, and returns it. By default, the input is treated as a string.

Syntax: input(prompt)

Example:

# Taking input from the user

name = input("Enter your name: ")
age = input("Enter your age: ")
print("Hello, " + name + "! You are " + age + " years old.")

Mutable and Immutable Objects

In Python, objects can be classified as mutable or immutable based on whether their state can be changed after creation.

Mutable Objects

Mutable objects can be changed after they are created. This means you can modify their content without creating a new object.

Example:

  • List
  • Dictionary
  • Set

Immutable Objects

Immutable objects cannot be changed after they are created. Any operation that seems to modify an immutable object actually creates a new object.

Example:

  • String
  • Tuple
  • Integer
  • Float
  • Frozen Set
